# Chapter 2: 分析方法论

## 为什么学这个？

投资界有两大流派，吵了一百多年：

- **技术分析派**说：「价格已经反映了一切信息，只需要看图表。」
- **基本面分析派**说：「价格经常是错的，要看公司的真实价值。」

谁对谁错？这不是一个你能靠「听别人说」来回答的问题。这一章你要**亲手实现两种方法**，用真实数据检验它们，然后自己判断。

## 学习目标

完成这一章后，你应该能够：

1. 手动计算并解释常见技术指标（MA、RSI、MACD、布林带）的含义
2. 阅读上市公司财报，计算关键财务比率
3. 用 DCF 模型对一家公司做基础估值
4. 区分「指标给出了信号」和「信号值得交易」之间的巨大鸿沟
5. 对「技术分析到底有没有用」形成自己的、有数据支撑的观点
