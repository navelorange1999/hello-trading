# 4.2 投资组合优化器

## 为什么做这个项目？

你已经在做资产配置了——黄金、股票、基金。但你的配置比例是怎么决定的？凭感觉？各买一点？

1952 年，哈里·马科维茨提出了现代投资组合理论（MPT），他证明了一个反直觉的结论：**把两个单独看都不那么好的资产组合在一起，整体表现可能比任何单个资产都好**。关键在于它们之间的「不相关性」。

这个项目让你亲手实现这个理论，然后用你自己的真实持仓来检验它。

## 已知领域

- 你已经在做资产配置（黄金、股票、基金）
- Chapter 1 的数据管道和统计基础（收益率、波动率、相关性）
- 基础的 numpy/pandas 操作

## 新领域

- 现代投资组合理论（马科维茨模型）
- 有效前沿的计算
- 各种配置方法的对比
- 将理论应用到你自己的投资组合

## Task 1: 计算多资产的收益率、波动率和相关性矩阵

**做什么**: 选择 5-10 个不同类别的资产（A 股、美股、黄金、债券基金、加密货币...），计算它们的历史收益率、波动率、两两相关性。

**思考提示**:
- 为什么你选了这些资产？它们覆盖了哪些「不同」的风险来源？
- 黄金和股票的相关性是多少？为什么这个数字在不同时期可能会变化？
- 高相关性（>0.8）的两个资产放在一起有分散风险的效果吗？
- 相关性矩阵中最让你惊讶的数字是哪个？为什么？

**验证**: 生成一张热力图展示相关性矩阵，每个数字你都能解释其经济含义。

📚 **关键资料**:
- 搜索 "correlation matrix heatmap matplotlib seaborn"
- 搜索 "asset class correlation" — 不同资产类别的典型相关性
- 注意：相关性不是恒定的，搜索 "correlation breakdown crisis" 了解危机期间相关性飙升的现象

---

## Task 2: 实现马科维茨均值-方差模型

**做什么**: 实现有效前沿（Efficient Frontier）的计算和可视化。找到最小方差组合和最大夏普比率组合。

**思考提示**:
- 有效前沿上的每一个点代表什么？前沿下方的点呢？
- 最小方差组合和最大夏普组合通常哪个更保守？
- 马科维茨模型的输入是什么？（预期收益率、波动率、相关性）这些输入可靠吗？
- 模型建议你把 80% 的钱投到某一个资产——你会照做吗？为什么不？（搜索 "estimation error portfolio optimization"）

**验证**: 画出有效前沿曲线，标注出关键组合（最小方差、最大夏普），每个资产的权重清晰可见。

📚 **关键资料**:
- 搜索 "Markowitz efficient frontier Python" — 理解计算方法
- 搜索 "scipy.optimize.minimize" — Python 中做优化的核心工具
- 搜索 "estimation error portfolio optimization" — 理解模型的实际局限性

---

## Task 3: 实现并对比多种配置方案

**做什么**: 实现以下配置方法，在同一张图上对比它们的历史表现：
1. 等权重（每个资产平均分配）
2. 最小方差（波动率最小的组合）
3. 风险平价（Risk Parity，每个资产贡献相同的风险）
4. 最大夏普（马科维茨最优组合）

**思考提示**:
- 等权重方法看似最「笨」，但它的表现经常不差。为什么？（搜索 "1/N portfolio"）
- 最大夏普组合在样本内是最优的，但在样本外呢？它和过拟合有什么关系？
- 风险平价的核心思想是什么？为什么桥水基金（Bridgewater）的全天候策略用的是这个方法？
- 如果你只能选一种方法用于你自己的真实投资，你会选哪个？为什么？

**验证**: 四种方法的收益曲线在同一张图上对比，附带关键指标（年化收益、波动率、夏普、最大回撤）。

📚 **关键资料**:
- 搜索 "risk parity portfolio explanation" — 风险平价的核心思想
- 搜索 "1/N portfolio DeMiguel" — 关于等权重组合惊人有效性的研究
- 搜索 "Bridgewater All Weather portfolio" — 全天候策略的基本架构

## Task 4: 用你自己的持仓做组合诊断

**做什么**: 把你现在真实持有的资产（黄金、股票、基金等）输入你的工具，分析当前的组合特征。

**思考提示**:
- 你的当前配置在有效前沿上的什么位置？是在前沿上还是前沿下方？
- 你的组合中，哪个资产贡献了最大的风险？可能不是你以为的那个。
- 你目前的配置有没有明显可以改进的地方？比如两个高度相关的资产占比都很大？
- 理论上的「最优配置」和你实际愿意执行的配置之间有差距吗？这个差距来自哪里？

**验证**: 产出一份你真实持仓的诊断报告，包含风险贡献分解、在有效前沿上的位置、改进建议。

---

## Task 5: 设计你的目标配置方案

**做什么**: 综合前面所有分析，为你自己设计一个资产配置方案。考虑你的目标（睡后收入）、风险承受能力、时间跨度。

**思考提示**:
- 你的目标是「睡后收入」，这意味着你需要什么样的特征？（稳定的现金流？长期增长？低波动？）
- 你能承受多大的短期亏损？如果账户一周内亏了 20%，你会坚持计划还是恐慌卖出？
- 配置方案应该多久调整一次？每天？每月？每年？调整的依据是什么？
- 你需要多少「闲钱」才应该开始认真投资？紧急备用金和投资资金的界限在哪里？

**验证**: 产出一份完整的配置方案文档：资产列表、目标比例、再平衡规则、你的理由。你不需要立刻执行它，但它应该是经过思考的。

---

## 完成标准

- **行为验证**: 你的优化器能对任意资产组合计算有效前沿，输出各种配置方案
- **解释验证**: 你能解释为什么分散投资有效（不只是「不要把鸡蛋放在一个篮子里」，而是数学上的解释）
- **迁移验证**: 如果有人给你一个配置方案，你能评估它的风险特征和改进空间

## 延伸思考

马科维茨因为投资组合理论获得了诺贝尔经济学奖。但有人问他自己的钱是怎么投的，他说「50% 股票、50% 债券」——基本就是等权重。

这个小故事揭示了一个深刻的道理：**理论上的最优和实践中的可行之间有鸿沟**。你在这个项目中应该既学会理论，也理解这个鸿沟。
