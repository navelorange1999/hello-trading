# Chapter 4: 风险管理与资产配置

## 为什么学这个？

华尔街有句老话：「市场可以保持非理性的时间，比你保持偿付能力的时间更长。」

你可能已经注意到一个现象：你身边（包括网上）炒股赚钱的人，往往只讲他们赚钱的那几次。但长期来看，大多数散户都在亏钱。为什么？

答案通常不是他们选错了股票，而是他们不懂风险管理：
- 赚了 50% 之后满仓加杠杆 → 一次回调就把利润全部吞掉
- 跌了 30% 不舍得止损 → 亏损从可控变成灾难性的
- 所有钱都押在一个方向 → 黑天鹅事件来了无处可逃

这一章是整个学习路线中**最容易被忽视但最重要**的部分。

## 学习目标

完成这一章后，你应该能够：

1. 量化一个投资组合的风险（不只是「感觉风险大」，而是具体数字）
2. 根据策略特征和你的风险承受能力，计算合理的仓位大小
3. 理解分散投资为什么有效（数学上的，不只是直觉上的）
4. 用现代投资组合理论优化你自己的资产配置
5. 对「该不该止损」「该不该加仓」这类问题有框架化的思考方式
