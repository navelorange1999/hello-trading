# Chapter 1: 市场运作原理与数据基础

## 为什么从这里开始？

你已经在买卖股票、基金和黄金了，但你有没有想过：当你点下「买入」的那一刻，到底发生了什么？你的订单去了哪里？价格是怎么形成的？为什么同一秒钟同一只股票会有不同的价格？

这一章要解决的核心问题是：**市场的底层机制是什么？**

大多数散户从来不思考这个问题，他们只关注「该买什么」。但如果你不理解市场本身的运作方式，你所有的分析都建立在一个你不了解的黑箱之上。

## 学习目标

完成这一章后，你应该能够：

1. 解释一个订单从提交到成交的完整生命周期
2. 理解买卖价差（bid-ask spread）的经济含义
3. 独立获取、清洗和存储市场数据
4. 用数据计算基本的统计指标（收益率、波动率、相关性）
5. 解释 K 线图上每一个元素的含义（因为你自己画过）
