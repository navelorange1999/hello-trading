# Chapter 3: 策略设计与回测

## 为什么学这个？

你在 Chapter 2 实现了各种分析指标，可能已经发现了一些「看起来能赚钱」的模式。但这里有一个残酷的事实：**人类大脑天生擅长在随机数据中「看到」规律**。

这就是回测存在的意义——它迫使你把模糊的直觉变成精确的规则，然后在历史数据上无情地检验。你会发现，大多数「看起来很美」的策略，一旦加入交易成本和滑点，就变得平庸甚至亏损。

这个发现本身就价值连城——它能帮你避免在真金白银上犯这些错误。

## 学习目标

完成这一章后，你应该能够：

1. 从零设计并实现一个回测引擎
2. 理解回测的常见陷阱（前视偏差、幸存者偏差、过拟合）
3. 评估一个策略的真实表现（不只是收益率，还有风险调整后的指标）
4. 区分「策略在历史上赚钱」和「策略在未来能赚钱」
5. 用数据解释什么是过拟合，为什么它如此危险
